## [Unreleased]

### Added
- Initial project setup with Debian 12 Bookworm base
- XFCE 4 desktop environment
- Firefox ESR web browser
- Network management tools (NetworkManager, WiFi firmware)
- Audio support (PulseAudio, ALSA)
- VMware tools with clipboard integration
- Development tools (Node.js, Git, curl, wget, nano, vim-tiny, htop)
- Google Gemini CLI integration
- Partition tools (GParted, parted, dosfstools, ntfs-3g)
- Dual boot support (BIOS/UEFI)
- Hybrid ISO format (disc and USB stick compatible)
- Auto-login for 'live' user
- Keyboard layout setup (English + Arabic with Alt+Shift toggle)
- Persistence support for saving changes across reboots
- Code of Conduct (Contributor Covenant)
- Contributing guidelines
- MIT License
